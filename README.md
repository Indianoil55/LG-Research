# 🛡️ LG Research - Advanced CF Bypass API

**LG Research** ek powerful tool hai jo Cloudflare (CF) protected websites se data extract karne ke liye banaya gaya hai. Yeh tool `undetected-chromedriver` ka use karke bot detection ko bypass karta hai.

### 🚀 Features
- **Auto-Bypass**: Cloudflare JS Challenge ko bypass karta hai.[cite: 2]
- **Custom API**: Apni khud ki API endpoints (e.g., `/lg-fetch`).[cite: 2]
- **Headless Support**: Server par bina UI ke chalta hai.[cite: 2]

### 🛠️ Installation
1. Repository clone karein: `git clone https://github.com/YourUsername/LG-Research.git`
2. Dependencies install karein: `pip install -r requirements.txt`
3. Server start karein: `python app.py`
